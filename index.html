IWT Mini Project

This GitHub Pages view is a static preview. The full app is a PHP file that requires a PHP + MySQL server and cannot run on Pages.

Run locally (recommended)

  1. Clone the repo or download MINI_PROJECT_IWT.php.
  2. Place it in your web root (e.g., htdocs), update DB creds in the $cfg array if needed.
  3. Start Apache/MySQL and open http://localhost/MINI_PROJECT_IWT.php.
Download PHP file

What’s inside

  • User auth (signup/login/logout) via PDO + MySQL
  • Dashboard, activity log, saved calculator history
  • Advanced calculator + converters, validator, DB insert demo
  • Event lab, hobby board, weekday helper, dark/light theme
